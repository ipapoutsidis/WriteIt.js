# Cursor.js
Cursor.js is lightweight javascript library for text-writing/type-writing effect.


# Installation
    - Download .zip file and extract cursor.js to your javascript folder and add
    '''
        <head>
            <script src="/path/to/cursor.js"></cursor.js>
        </head>
    '''
